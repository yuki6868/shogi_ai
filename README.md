# せめぎ合い将棋AI

子供向けの「せめぎ合い」を重視した教育用将棋AIです。

普通の将棋AIのように最強を目指すのではなく、

- あと少しで勝てそう
- 惜しかった
- 次は勝ちたい

と思える対局体験を目指しています。

現在は、

- 将棋盤UI
- 駒移動
- 成り
- 持ち駒
- 最低限の反則チェック
- Pythonバックエンド連携

まで実装しています。

## フロントエンド

HTML側は、盤面操作と表示だけを担当します。

担当内容:

- 将棋盤表示
- 駒移動
- 持ち駒
- 成り
- ターン管理
- Pythonへの盤面送信

AIロジック、接戦評価、棋力推定はHTML側では管理しません。

---

## フロントエンド起動方法

`frontend/index.html` をブラウザで開きます。

```text
frontend/index.html
```

---

## バックエンド

Python側では、AIに関する処理を担当します。

担当内容:

- AIの思考
- 棋力推定
- 接戦演出
- 合法手生成
- 評価関数
- ヒント生成

---

## バックエンド起動方法

### 1. backendへ移動

```bash
cd backend
```

### 2. ライブラリをインストール

```bash
pip install -r requirements.txt
```

### 3. FastAPIを起動

```bash
uvicorn main:app --reload
```

起動後、以下のURLでバックエンドが待ち受けます。

```text
http://127.0.0.1:8000
```

## コンセプト

このAIの目的は、単に強い手を指すことではありません。

目的は、

```text
子供が「あと少しで勝てそう」と思える対局を作ること
```

です。

そのために、Python側では以下のような処理を実装していく予定です。

- 強すぎる手を避ける
- 小さな逆転チャンスを残す
- 接戦を維持する
- 子供の成長に合わせて強さを変える
- 負けたあとに学習ポイントを1つだけ提示する

---

## 開発方針

HTML側は盤面UIに集中します。

Python側はAIロジックに集中します。

```text
HTML:
盤面表示、操作、入力

Python:
AI、評価、せめぎ合い、学習支援
```

この分離により、あとからAIロジックだけを改善しやすくします。
